# Sneakers Market App

## Project Description
The Sneakers Market App is a full-stack web application that allows users to browse sneaker products with details such as brand, price, and images. The project demonstrates frontendâ€“backend integration using REST APIs.

## Tech Stack
Frontend: React JS, HTML, CSS, Axios  
Backend: Node.js, Express.js  
Database: MongoDB  

## Features
- Display sneaker products
- Fetch data using REST APIs
- MongoDB data storage
- Clean and responsive UI
- Separation of frontend and backend

## Project Structure
frontend/ contains the React application  
backend/ contains the Node.js and Express API  

## How to Run the Project

### Backend
cd backend
npm install
node server.js

Server runs on: http://localhost:5000

### Frontend
cd frontend
npm install
npm start

Application runs on: http://localhost:3000

## Application Flow
User opens the frontend application  
Frontend requests data from backend API  
Backend fetches sneaker data from MongoDB  
Data is returned and displayed to the user
